### 기능 구현 목록 작성

* 메인 UI 작성
  * 시작 문구가 출력된다.
* 코치의 이름을 입력받는 UI 작성
  * 잘못된 입력일 경우 예외가 발생한다.
    * , 로 구분된다.
    * 이름은 2글자에서 4글자이다.
    * 최소 2명, 최대 5명이다.
* 각 코치 별로 못먹는 메뉴를 입력받는 UI 작성
  * 잘못된 입력일 경우 예외가 발생한다.
    * 최소 0개, 최대 2개이다.
* 요일별로 카테고리를 추천하는 기능
  * 매일 음식의 카테고리가 선택되어야 한다.
    * 한주에 같은 카테고리가 2개를 초과할 수 없다.
* 메뉴를 추천하는 기능 작성
  * 중복되는 메뉴가 존재하면 안된다.
* 메뉴를 추천하는 UI 작성
